# Smart Campus App (iOS)

## Overview
A scalable iOS application designed for campus services using modern SwiftUI architecture.

---

## Features
- SwiftUI + MVVM architecture
- Firebase Authentication
- Google OAuth login
- Real-time database updates
- Deep linking support
- External service integration

---

## Architecture
View → ViewModel → Firebase Backend

---

## Tech Stack
- SwiftUI
- Firebase
- MVVM pattern
- iOS SDK

---

## Key Highlights
- Secure authentication system
- Real-time data synchronization
- Modular and scalable UI design
